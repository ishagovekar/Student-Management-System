# 🎓 Student Management System

A desktop-based Student Management System built using Python, Tkinter, and SQLite.

## 🚀 Features

- Add Student
- View Student Records
- Delete Student
- Update Student
- SQLite Database Integration
- GUI Interface using Tkinter

## 🛠️ Technologies Used

- Python
- Tkinter
- SQLite3
